*** HƯỚNG DẪN DI CHUYỂN CODE HTML SANG ***
1. Mở file thư mục ui/Header/index.jsx, chuyển đổi đoạn HTML Navbar vào thẻ return. Tương tự cho Footer.
2. Layout/ClientLayout/index.jsx: Sẽ Import Header và Footer vào, đặt {children} ở giữa.
3. Các file bên trong "pages": Mở code trang tương ứng (.html) -> Chép đoạn HTML từ phần nội dung thân web (nằm sau navbar, trước footer) dán vào thẻ return của index.jsx.
4. Đổi toàn bộ chữ "class=" thành "className=". Đóng tất cả thẻ đơn như <img> thành <img /> hoặc <input />.
5. Xóa bỏ thẻ <script> dưới cùng vì bên React thì đã có cách tổ chức hiệu ứng DOM và click tự động thông qua function Component.

*** HƯỚNG DẪN MỞ RỘNG (Dùng khi dự án lớn hơn) ***
Sau này bạn hãy chủ động tạo thêm các thư mục:
- src/hooks/: Nơi để các custom hook tự viết (useFetch, useAuth...).
- src/services/ (hoặc src/api/): Nơi gọi API Backend (axios, fetch) thay vì gọi trực tiếp lên giao diện.
- src/utils/: Chứa các hàm hỗ trợ chung (format tiền tệ 500.000đ, xử lý định dạng ngày tháng...).
- src/constants/: Chứa các biến cố định (URL API chuẩn, danh sách danh mục xe...).
